Icons by Aaron Pinero

Over the years I have created several custom icons. I did this to avoid using a very large icon font like FontAwesome when all I needed was a handful of icons.

There are 30 icons in this collection. The icons are packaged as web fonts. There are light, regular, and bold weight versions of the icons. Each icon font is generated using the svgtofont npm package.

The example icons below are shown at 175% normal size. I have also included the equivalent free FontAwesome icons for comparison.
